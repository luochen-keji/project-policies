
# 落宸写作助手
一款简约高效、安全加密的码字写作工具，专为小说作者、文字创作者打造。
## ✨ 软件介绍
落宸写作助手 是由开发者独立开发的轻量化写作辅助软件，主打**本地加密存储**、无强制联网，专注提供安静、稳定的码字创作环境。<br>
当前版本：完全免费<br>
数据方案：全量本地加密存储，无需登录、不上传隐私内容<br>
未来规划：后续迭代将可选新增账号登录、跨设备云同步、增值付费功能<br>
核心定位：网文写作、日常文稿、大纲编辑、内容管理<br>
## 🚀 核心功能
📝 轻量化编辑器，专注码字创作<br>
🔐 本地 AES 加密存储，保护原创文稿安全<br>
💾 本地文档管理、分类、导出、备份<br>
🎨 自定义主题、界面布局、排版设置<br>
🧩 纯净无推送、无广告、无后台偷跑联网<br>

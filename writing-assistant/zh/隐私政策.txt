
落宸写作助手 隐私政策
生效日期：2026年4月27日
落宸写作助手（以下简称“本软件”“我们”）尊重并保护所有使用本软件用户（以下简称“您”）的个人隐私与数据安全。本隐私政策依据《中华人民共和国个人信息保护法》《网络安全法》《网络数据安全管理条例》等相关法律法规制定，清晰说明本软件当前及未来迭代后，对您信息的收集、使用、存储、保护等相关事宜，同时明确您享有的隐私权利，请您仔细阅读并理解。
本隐私政策将随软件功能迭代（如新增登录、云同步功能）进行修订，修订后将在软件内显著位置及GitHub隐私政策页面公示，公示期满后生效；您继续使用本软件即视为同意修订后的全部条款。若您不同意修订内容，可停止使用本软件，我们将不再处理您的任何相关信息。
一、核心隐私原则
1.  最小收集原则：仅收集为提供软件功能所必需的信息，不超范围收集无关个人信息，不通过误导、欺诈、胁迫等方式取得您的同意；
2.  本地优先原则：当前版本所有用户数据均存储于您的设备本地，不主动上传至任何服务器，我们无法获取、访问您的本地数据；
3.  加密保护原则：本地数据采用行业标准加密算法存储，保障您的文档内容安全，防止被篡改、泄露；
4.  迭代透明原则：未来新增登录、云同步等功能时，将明确告知信息收集范围、用途及存储方式，征得您的明示同意后再开展相关服务，不强制您使用新增功能；
5.  自主可控原则：您对自身的文档数据拥有完全控制权，可随时查看、编辑、删除，数据的存储与删除均由您自主操作。
二、当前版本（无登录、无云同步，免费）信息处理说明
2.1 信息收集
本版本为免费版本，无需注册、无需登录，不收集任何个人身份信息（包括但不限于姓名、手机号、身份证号、邮箱、设备唯一标识、地理位置等），不内置任何第三方统计、追踪SDK，不获取您设备的任何敏感权限（如通讯录、相册、麦克风、摄像头等）。
仅在您使用软件过程中，生成并存储以下本地数据（全程仅保存在您的使用设备中，我们无法获取、访问或控制）：
1.  您在软件内创建、编辑、保存的文稿、笔记、模板、格式设置等内容（以下统称“用户文档”）；
2.  软件本地配置信息（如字体、主题、界面布局、操作习惯等），仅用于优化您的使用体验，不涉及任何隐私内容；
3.  软件运行日志（仅在您主动提交问题反馈时，可选择附带日志，日志仅包含软件运行状态，不含任何个人信息及用户文档内容，仅用于排查故障、优化功能）。
2.2 信息使用
收集的本地数据仅用于以下用途，不用于任何其他目的，不进行任何信息分析、共享或商业化利用：
1.  提供核心写作功能（如文档编辑、格式排版、本地导出、保存备份等），确保软件正常运行；
2.  记忆您的使用习惯（如字体设置、主题偏好），优化软件操作体验，提升功能响应效率；
3.  若您主动提交反馈，附带的运行日志仅用于修复软件Bug、改进功能。
2.3 信息存储与保护
1.  存储方式：所有用户数据均采用本地加密存储，仅存储在您的使用设备（电脑/手机）中，不上传至云端或我们的服务器，完全由您自主管控；
2.  存储期限：数据存储期限由您自行决定，您可随时在软件内删除单篇用户文档、清空全部本地数据，删除后数据仅在您的设备中可通过本地备份恢复（若您开启设备本地备份），我们无法恢复或获取已删除数据；
3.  安全保护：我们采用行业标准的加密技术（如AES加密算法）保护本地数据，防止数据被篡改、泄露；但互联网及设备本身无绝对安全，建议您定期备份本地文档，妥善保管您的设备，避免设备丢失、被盗导致数据泄露，因设备自身问题导致的数据损失，我们不承担责任；
4.  数据销毁：您卸载本软件时，设备内存储的所有用户数据将随软件卸载一并删除（若您未提前备份），我们不会留存任何相关数据。
2.4 信息共享与披露
当前版本，我们不收集、不存储您的任何个人信息及用户文档数据，因此不存在信息共享、转让、公开披露的情况；
唯一例外情形：若因法律法规要求、政府部门依法履职需要，我们将配合提供相关信息（但当前版本无任何可提供的信息），且会在法律允许范围内，最大限度保护您的隐私权益。
三、未来迭代功能（可能新增登录、云同步、收费服务）信息处理说明
未来本软件将根据迭代计划，可能新增登录功能（如账号登录、第三方登录）、云同步功能，同时可能推出付费增值服务，相关信息处理将遵循以下规则，具体以届时公示的修订版隐私政策为准，确保符合合规要求：
3.1 登录功能相关
1.  登录功能为可选功能，您可自主选择是否注册登录，登录仅用于使用云同步等增值功能，不强制要求登录，不登录不影响当前版本核心免费功能的使用；
2.  若您选择注册登录，我们可能收集您的账号（如手机号、邮箱）、密码（采用加密存储方式），仅用于身份验证，不关联其他个人信息，不用于其他用途；
3.  若您选择第三方登录（如微信、QQ等），我们仅获取第三方平台授权的基本信息（如昵称、头像），不获取第三方平台的其他隐私信息，授权范围以第三方平台公示为准，您可随时在第三方平台撤销授权；
4.  我们将采取严格的安全措施保护您的登录信息，防止账号被盗、信息泄露，同时您需妥善保管自身账号密码，因您自身操作不当导致的账号安全问题，我们不承担责任。
3.2 云同步功能相关
1.  云同步功能为可选功能，您可自主选择是否开启，开启后您的用户文档将通过安全传输协议（TLS）加密传输并存储至我们的云端服务器，仅用于跨设备同步您的文档，方便您在不同设备间使用；
2.  云端存储的数据将采用加密存储方式，仅您本人可通过登录账号访问、管理，我们不会查看、篡改、泄露您的云端文档，不会用于广告、推荐或第三方共享；
3.  云端数据存储期限与您的账号有效期一致，您可随时关闭云同步、删除云端数据，或注销账号（注销后云端数据将全部删除，无法恢复）；
4.  我们将采取行业标准的安全措施保护云端服务器，限制访问云端数据的人员范围，定期进行安全检测，防止数据泄露、丢失、篡改，同时会按照法律法规要求，将境内收集的用户信息存储于中国境内，如需跨境传输，将提前告知您并征得您的同意。
3.3 收费服务相关
未来推出的收费服务，仅涉及功能权限的开通（如云同步扩容、专属模板等），不改变信息收集、使用、存储的核心规则，不额外收集您的个人信息；
收费服务的相关规则（如价格、服务期限、退费政策）将单独公示，您可自主选择是否购买，不购买不影响当前免费功能的使用。
四、您的权利
您可随时行使以下权利，我们将在合理期限内响应您的诉求：
1.  访问与导出权：在软件内查看、导出您的所有用户文档及本地配置信息；
2.  删除权：删除单篇用户文档、清空全部本地数据，或在未来开启云同步后，删除云端数据；
3.  撤回授权权：未来开启登录、云同步功能后，可随时关闭相关功能、注销账号，撤回信息收集授权；
4.  知情权：了解本软件信息收集、使用、存储的相关规则，以及未来功能迭代后的信息处理变化；
5.  投诉与建议权：若您对隐私保护有任何疑问、投诉或建议，可通过本政策末尾的联系方式与我们联系，我们将在15个工作日内回复并处理。
五、未成年人保护
本软件面向18周岁及以上用户。未成年人需在监护人同意下使用本软件，我们不主动收集未成年人的任何个人信息；若监护人发现未成年人误使用本软件并产生相关数据，可联系我们协助删除，我们将积极配合。
